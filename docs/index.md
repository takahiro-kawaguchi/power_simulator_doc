# power_simulator_doc

power_simulator の公式ドキュメントです  
power_simulator では主に，研究者・技術者向けに電力系統を用いたシミュレーション環境を提供しています  
IEEE68bus system などの既存のシステムや既存のコントローラを利用した応答シミュレーションや，新たなネットワーク・機器・制御器を作成してそれぞれのニーズに合わせたシミュレーションを行うことができます
